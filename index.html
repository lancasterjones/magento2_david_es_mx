Language Pack / Paquete de lenguaje

This package has most of translation for latin spanish (es_MX) just frontend scope.

Este paquete contiene la mayoria de las traducciones para el español latino (es_MX) solo aplica a la vista frontend

Installation / Instalacion

Using modman / Usando momdan:

  • go to magento 2 root directory / ve a la raiz del directorio de magento 2
  • modman init
  • modman clone https://github.com/JDavidVR/magento2_david_es_mx.git

Manually / Manualmente:

  • Download compressed files / Descarga el archivo compreso
  • Copy extension app folder to magento root directory / Copia el folder app de la extension al directorio raiz de magento

IMPORTANT: Don't forget to change store language on backend going to Store > Configuration > Configuration > General > Locale, select Spanish Mexico

IMPORTANTE: No te olvides de cambir el idiona en el bavkend yendo a Store > Configuration > Configuration > General > Locale, selecciona Spanish Mexico

Translation issues / Problemas de traduccion

In case you found any bad or missing translation, please notify or do the correction and request a pull on github. Thank you for improving this package.

En caso de encontrar alguna traduccion incorrecta o flatante, por favor notificar o hacer la correction y solicitar un pull en github. Muchas gracis por ayudar a mejorar este paquete
